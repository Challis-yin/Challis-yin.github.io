发表于2021-04-03|更新于2021-04-03|杂谈博客搭建

前言

现在已经不是一个个人博客兴盛的年代了，就想写信寄信都不再贴上邮票而是直接快递填单一样，个人博客可以说已经完全被微博、知乎、各类博客网站甚至不可见的朋友圈代替了，但是作为独立思考的个体，一个个人博客无疑有以下的优点：

  • 个人对话的平台
  • 持续学习的动力
  • 提高将事情讲清楚的能力
  • 记录成长讨论反思
  • 个人的精神家园

以下将以本文发布博客的搭建过程为例，解释如何快速搭建一个类似的博客

  • 一个GitHub账号
